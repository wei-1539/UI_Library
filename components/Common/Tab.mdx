# Tab

頁籤元件，支援三種視覺樣式、鍵盤導航、個別禁用，以及 `v-model` 雙向綁定。

## Item 結構

```ts
interface TabItem {
    label: string       // 顯示文字
    value: string       // 唯一識別值（綁定用）
    disabled?: boolean  // 是否禁用
    icon?: string       // icon 名稱（對應 AtomIcon）
}
```

## 鍵盤操作

<table>
  <thead>
    <tr>
