# Accordion 手風琴

由 `Accordion`（容器）與 `AccordionItem`（項目）組合使用的折疊面板元件。
<br/>支援單選 / 多選、受控 / 非受控、停用，也可以將 `AccordionItem` 單獨使用。

---

## 元件結構

```
Accordion        ← 狀態管理容器，透過 provide 往下傳
  └── AccordionItem  ← 單一折疊項目，透過 inject 接收狀態
  └── AccordionItem
  └── AccordionItem
```

## 受控 vs 非受控

<table>
  <thead>
    <tr>
      <th>模式</th>
      <th>寫法</th>
      <th>狀態在哪</th>
      <th>適用情境</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>受控</td>
      <td><code>v-model="open"</code></td>
      <td>父層的變數</td>
      <td>需從外部控制開關，例如點擊頁面其他按鈕自動展開特定項目</td>
    </tr>
    <tr>
      <td>非受控</td>
      <td>不傳 v-model</td>
      <td>元件內部 ref</td>
      <td>只需要折疊功能，不需要外部感知狀態</td>
    </tr>
  </tbody>
</table>
