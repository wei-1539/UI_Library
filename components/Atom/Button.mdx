# Button

通用按鈕元件，會根據傳入的 props 自動切換底層標籤：

- 預設 → `<button>`
- 傳入 `href` → `<NuxtLink>`（站內路由）
- `disabled` 為 `true` → 強制回退成 `<button>`，連結失效

## 注意事項

- `text` prop 會被 **slot 內容覆蓋**，當你用 `<Button>自訂內容</Button>` 時，`text` 失效
- `target` 控制是否開新分頁，底層對應 `_blank` / `_self`，只在渲染成 `NuxtLink` 時有效
- `disabled + href` 同時傳入時，元件**不會**渲染成連結，disabled 優先
