# Breadcrumb

麵包屑導航元件，根據傳入的 `navList` 陣列自動產生路徑列表：

- 有 `to` 的項目 → 渲染成 `<AtomButton>`（可點擊連結）
- 沒有 `to` 的項目 → 渲染成純文字 `<span>`，最後一項自動降低透明度表示「目前頁」
- 每個項目之間以 `separator` 分隔

## 注意事項

- 最後一個沒有 `to` 的項目會自動加上 `opacity-50`，表示「目前所在頁」
- `separator` 不會出現在最後一個項目之後
- `icon` 和 `iconOnly` 需搭配有 `to` 的項目才有效（因為純文字項目不渲染 `AtomButton`）
