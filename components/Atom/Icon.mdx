# Icon

基於 SVG Sprite 的 Icon 元件，透過 `vite-plugin-svg-icons` 將所有 SVG 打包成 sprite，使用時只需傳入名稱即可。

## 使用方式

```vue
<AtomIcon name="add" size="md" variant="primary" />
```

## 新增 Icon

把 `.svg` 檔放進 `assets/icons/` 資料夾，Gallery 會自動出現，不需要手動註冊。

```
assets/
└── icons/
    ├── add.svg        → name="add"
    ├── arrow-down.svg → name="arrow-down"
    └── your-icon.svg  → name="your-icon"  ← 直接放進來就好
```
